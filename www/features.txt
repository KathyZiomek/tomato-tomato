Pomodoro Timer

Needs
- counts down from a time set by a user
- user can set how long they want breaks to be

Wants
- have alarm go off when each time segment finishes
- user can set how many work sessions they want before a long break
- user can set if they want long breaks
- user can set how long in total they want to work
	which will evenly distribute the work and break times in the set period

Nice to Haves
- store study time on local device
- pause music applications (like Spotify) when alarm goes off
- customizable alarm
- light mode/dark mode and other customizations
